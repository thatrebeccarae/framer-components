# Minimal Light Theme

A bright, clean design with dark text. Simple and airy.

![Minimal Light Preview](../assets/wizard-screenshots/theme-minimal-light.png)

## Color Palette

| Element | Color | Format |
|---------|-------|--------|
| Background | `#FFFFFF` | Pure white |
| Text (Primary) | `#1A1A1A` | Near black |
| Text (Secondary) | `rgba(26, 26, 26, 0.6)` | 60% black |
| Card Background | `rgba(0, 0, 0, 0.05)` | 5% black |
| Card Hover | `rgba(0, 0, 0, 0.1)` | 10% black |
| Icons | `#1A1A1A` | Near black |
| Icon Hover | `rgba(26, 26, 26, 0.6)` | 60% black |
| Accent (Pulse) | `rgb(59, 130, 246)` | Blue |

## Typography

| Element | Font | Weight | Size |
|---------|------|--------|------|
| Username | Inter | SemiBold (600) | 16px |
| Bio | Inter | Medium (500) | 14px |
| Link Name | Inter | SemiBold (600) | 15px |
| Link Description | Inter | Medium (500) | 13px |

## Component Settings

### LinksHeader
```
usernameColor: "#1A1A1A"
bioColor: "rgba(26, 26, 26, 0.6)"
```

### SocialIconLinks
```
iconColor: "#1A1A1A"
hoverColor: "rgba(26, 26, 26, 0.6)"
```

### LinkCard
```
backgroundColor: "rgba(0, 0, 0, 0.05)"
hoverBackgroundColor: "rgba(0, 0, 0, 0.1)"
textColor: "#1A1A1A"
descriptionColor: "rgba(26, 26, 26, 0.6)"
borderRadius: 12
```

### LinksContainer
```
backgroundColor: "#FFFFFF"
maxWidth: 480
paddingVertical: 40
paddingHorizontal: 20
gap: 18
```

## Page Setup in Framer

1. Select the page canvas
2. In the right panel, find "Fill"
3. Enter: `#FFFFFF`

## Best For

- Clean, modern brands
- Professional services
- Photography portfolios (content focus)
- Daytime/bright environment viewing
- Corporate/business use

## Accessibility Notes

- Contrast ratio: ~17:1 (excellent)
- Familiar light mode pattern
- Good for printing
