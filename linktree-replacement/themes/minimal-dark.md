# Minimal Dark Theme

A clean, dark theme with white text. Professional and sleek.

![Minimal Dark Preview](../assets/wizard-screenshots/theme-minimal-dark.png)

## Color Palette

| Element | Color | Format |
|---------|-------|--------|
| Background | `rgb(15, 14, 14)` | Nearly black |
| Text (Primary) | `#FFFFFF` | White |
| Text (Secondary) | `rgba(255, 255, 255, 0.7)` | 70% white |
| Card Background | `rgba(255, 255, 255, 0.1)` | 10% white |
| Card Hover | `rgba(255, 255, 255, 0.2)` | 20% white |
| Icons | `#FFFFFF` | White |
| Icon Hover | `rgba(255, 255, 255, 0.7)` | 70% white |
| Accent (Pulse) | `rgb(255, 85, 136)` | Hot pink |

## Typography

| Element | Font | Weight | Size |
|---------|------|--------|------|
| Username | Inter | SemiBold (600) | 16px |
| Bio | Inter | Medium (500) | 14px |
| Link Name | Inter | SemiBold (600) | 15px |
| Link Description | Inter | Medium (500) | 13px |

## Component Settings

### LinksHeader
```
usernameColor: "#FFFFFF"
bioColor: "rgba(255, 255, 255, 0.7)"
```

### SocialIconLinks
```
iconColor: "#FFFFFF"
hoverColor: "rgba(255, 255, 255, 0.7)"
```

### LinkCard
```
backgroundColor: "rgba(255, 255, 255, 0.1)"
hoverBackgroundColor: "rgba(255, 255, 255, 0.2)"
textColor: "#FFFFFF"
descriptionColor: "rgba(255, 255, 255, 0.7)"
borderRadius: 12
```

### LinksContainer
```
backgroundColor: "rgb(15, 14, 14)"
maxWidth: 480
paddingVertical: 40
paddingHorizontal: 20
gap: 18
```

## Page Setup in Framer

1. Select the page canvas (click on empty area)
2. In the right panel, find "Fill"
3. Enter: `rgb(15, 14, 14)`

## Best For

- Professional portfolios
- Tech/dev personal brands
- Minimalist aesthetics
- OLED-friendly (true blacks)
- Evening/night viewing

## Accessibility Notes

- Contrast ratio: ~15:1 (excellent)
- Works well in dark environments
- Reduced eye strain
