# Step 3: Theme Selection

## What We'll Do

Choose a color theme for your link page.

## Before We Start

- [ ] Project created in Framer
- [ ] On your project canvas

## Available Themes

We have 6 pre-built themes. Preview each and pick your favorite:

### Minimal Dark
Clean, dark background with white text.
- Background: `rgb(15, 14, 14)`
- Text: `#FFFFFF`
- Cards: `rgba(255, 255, 255, 0.1)`
- Best for: Professional, sleek look

### Minimal Light
Bright, clean design with dark text.
- Background: `#FFFFFF`
- Text: `#1A1A1A`
- Cards: `rgba(0, 0, 0, 0.05)`
- Best for: Clean, airy feel

### Gradient Sunset
Warm orange-pink gradient background.
- Background: Linear gradient from `#FF6B6B` to `#FFE66D`
- Text: `#FFFFFF`
- Cards: `rgba(255, 255, 255, 0.2)`
- Best for: Creative, warm personality

### Gradient Ocean
Cool blue-teal gradient.
- Background: Linear gradient from `#667eea` to `#764ba2`
- Text: `#FFFFFF`
- Cards: `rgba(255, 255, 255, 0.15)`
- Best for: Calm, professional creatives

### Playful Colorful
Bright, energetic colors.
- Background: `#FFE4E1`
- Text: `#333333`
- Accent: `#FF69B4`
- Best for: Fun, youthful brands

### Professional Neutral
Muted, corporate-friendly.
- Background: `#F5F5F5`
- Text: `#2C3E50`
- Cards: `#FFFFFF` with border
- Best for: Business, consultants

## Choosing Your Theme

### Questions for You

1. What's the vibe of your brand?
   - Professional/Corporate → Minimal Dark, Professional Neutral
   - Creative/Artistic → Gradient themes, Playful
   - Personal/Casual → Any that resonates

2. Do you have existing brand colors?
   - If yes, you can customize any theme
   - If no, pick a pre-built theme

3. Light or dark mode?
   - Dark: Minimal Dark, Gradients
   - Light: Minimal Light, Professional Neutral, Playful

### Setting the Background

1. Click on the canvas (not any element)
2. In the right sidebar, find "Fill"
3. For solid color: Paste the background color
4. For gradient:
   - Click the color picker
   - Switch to "Gradient" mode
   - Add your gradient colors
