# Overview: Build Your Link Page

Welcome! This guide will help you create a beautiful, private link-in-bio page using Framer - no coding required.

## What You'll Build

A personalized link page with:
- Your profile photo and bio
- Social media icons
- Clickable link cards
- Optional analytics

**Time needed:** About 30-60 minutes for your first setup

## What You'll Need

- [ ] An email address (for Framer account)
- [ ] A profile photo (square works best)
- [ ] Your social media URLs
- [ ] A list of links you want to share

## How This Guide Works

This is a **step-by-step wizard** designed to work with Claude or other AI assistants. Each step includes:

1. **What we'll do** - Brief explanation
2. **Questions for you** - Information we need
3. **Instructions** - Step-by-step actions
4. **Checkpoint** - Verification that it worked

## Why Framer Instead of LinkTree?

| Feature | LinkTree | Your Framer Page |
|---------|----------|------------------|
| Monthly cost | $5-24/month for premium | Free |
| Custom domain | Paid feature | Free with any domain |
| Full customization | Limited | Complete control |
| Your data | They own it | You own it |
| Analytics | Their dashboard | Your Google Analytics |
| Remove branding | Paid feature | Always yours |

## Privacy
