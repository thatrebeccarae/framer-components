# Step 2: Project Setup

## What We'll Do

Create a new Framer project for your link page.

## Before We Start

- [ ] Logged into Framer
- [ ] On the Framer dashboard

## Instructions

### 1. Create New Project

From the dashboard:
1. Click **"New Project"** or the **"+"** button
2. Select **"Blank"** (not a template)
3. Your project opens in the editor

### 2. Name Your Project

1. Click on "Untitled" in the top-left corner
2. Type a name like "My Links" or "Link Page"
3. Press Enter

### 3. Set Up the Page

By default, you have a "Home" page. We'll use this for your links.

1. In the left sidebar, you'll see "Home" under Pages
2. Click on it to select it
3. You're ready to design!

### 4. Set Page Background (Optional)

To set a dark background:
1. Click anywhere on the empty canvas (not on any element)
2. In the right sidebar, find "Fill" or "Background"
3. Click the color box
4. Enter: `rgb(15, 14, 14)` for dark mode
   - Or choose your preferred color

### 5. Understand the Interface

Key areas you'll use:
- **Left sidebar:** Pages, layers, and assets
- **Center canvas:** Where you design
- **Right sidebar:** Properties for selected elements
- **Top toolbar:** Tools and view options

## Checkpoint

You should have:
- [ ] A new project with a custom name
- [ ] A single "Home" page
- [ ] The canvas is visible and ready

## Adding Code Components

We'll add the link components in later steps. For now, you can preview how to add code:

1. In the left sidebar, click **"Assets"** tab
2. Click **"+"** (Add)
3. Select **"Code Component"**
4. This creates a new component file

Don't worry about this yet - we'll do it step by step.
