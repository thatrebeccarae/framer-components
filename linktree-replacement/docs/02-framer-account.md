# Step 1: Create Framer Account

## What We'll Do

Create a free Framer account. This is where your link page will live.

## Before We Start

Make sure you have:
- [ ] An email address
- [ ] About 5 minutes

## Instructions

### 1. Go to Framer

Open your browser and go to: **https://framer.com**

### 2. Click "Start for Free"

You'll see a button in the top right corner. Click it.

### 3. Choose Sign-Up Method

You can sign up with:
- **Google** (fastest)
- **Email and password**

Pick whichever you prefer.

### 4. Complete Sign-Up

If using email:
1. Enter your email
2. Create a password
3. Click "Create Account"
4. Check your email for verification link
5. Click the verification link

If using Google:
1. Select your Google account
2. You're done!

### 5. Skip the Tutorial (Optional)

Framer may show you an onboarding tutorial. You can:
- Complete it to learn the basics
- Skip it and follow this guide instead

## Checkpoint

You should now see the Framer dashboard with:
- [ ] Your account name in the corner
- [ ] A "New Project" or "Create" button
- [ ] An empty or starter project list

## Common Issues

### "Email already in use"
You may already have an account. Try:
1. Click "Log In" instead
2. Use "Forgot Password" if needed

### Verification email not arriving
1. Check your spam folder
2. Wait a few minutes
3. Click "Resend" if available
4. Try a different email

### Page won't load
1. Try a different browser (Chrome recommended)
2. Disable ad blockers temporarily
3. Clear your cache
